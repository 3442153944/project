# 🗂️ 私有云系统说明文档 (v2026.03)

## 📖 项目简介
本项目为 **多端统一的私有云平台**，支持文件的自动同步、预览、在线编辑与系统监控等核心功能。  
主要目标是实现 **跨端文件管理与实时监控**，同时保障数据的隐私性与传输安全性。  
**2026年新增：** 全面适配 **HarmonyOS NEXT** 生态，支持华为全场景设备。

---

## 🚀 核心功能

### 🔄 自动文件同步
- **多端实时同步**：支持 Android、Windows、Web 及 **HarmonyOS** 客户端。
- **高效传输**：支持增量同步与断点续传，本地文件变更自动检测。
- **版本控制**：提供文件版本历史记录与一键回滚功能。

### 👀 文件预览与在线编辑
- **流媒体预览**：支持图片 (JPG/PNG)、文档 (PDF/TXT) 及音视频文件的在线流式播放。
- **在线编辑器**：
  - 内置 Word / Excel / Markdown 网页端编辑器。
  - 支持多端协作模式与修改记录追踪。

### 🧠 系统与网络监控
- **服务器监控**：实时查看 CPU 负载、内存占用、磁盘空间及网络流量图表。
- **内网安防**：支持局域网内摄像头视频实时查看，集成录制与历史回放。

## ⚙️ 技术栈

| 模块 | 技术实现 | 说明 |
| :--- | :--- | :--- |
| **后端服务** | **Go** `v1.25.3` (Gin + GORM) | 高并发逻辑处理中心 |
| **数据库** | **PostgreSQL** `v18` | 核心业务数据存储 |
| **缓存/队列** | **Redis** `v5.0.10` | 状态缓存与监控数据暂存 |
| **反向代理** | **NGINX** `v2.22.0` | 静态资源分发与 SSL 安全加速 |
| **Android 端** | **Kotlin** (`Android 10+`) | 移动端原生应用 |
| **Windows 端** | **Tauri** (`v2.0.0+`, `Rust`) | 桌面客户端 |
| **Web UI** | **Vue 3** + TS + Element Plus | 响应式管理后台 |
| **HarmonyOS 端** | **ArkTS** + **ArkUI** (NEXT) | 华为全场景适配 |
