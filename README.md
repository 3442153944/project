# 🗂️ 私有云系统说明文档

## 📖 项目简介
本项目为 **多端统一的私有云平台**，支持文件的自动同步、预览、在线编辑与系统监控等核心功能。  
主要目标是实现 **跨端文件管理与实时监控**，同时保障数据的隐私性与传输安全性。  

---

## 🚀 核心功能

### 🔄 自动文件同步
- 多设备间实时文件同步（Android / Windows / Web）
- 支持增量同步与断点续传
- 本地文件变更自动检测并推送至云端
- 提供版本历史与回滚功能

### 👀 文件预览与在线编辑
- 支持常见文件类型在线预览：
  - 图片（JPG、PNG、GIF）
  - 文档（PDF、DOCX、XLSX、TXT）
  - 视频与音频文件
- 内置 **在线编辑器**：
  - Word / Excel / Markdown 可直接在浏览器中编辑
  - 支持协作模式与修改记录

### 🧠 系统与网络监控
- **系统监控**
  - 查看服务器实时 CPU、内存、磁盘使用率
  - 网络流量与请求统计图表
- **内网监控视频**
  - 支持局域网内摄像头视频实时查看
  - 支持录制与回放

---

## ⚙️ 技术栈

| 模块 | 技术 |
|------|------|
| **后端服务** | Go `v1.25.3` (`windows/amd64`) |
| **数据库** | PostgreSQL `v18` |
| **缓存与消息队列** | Redis `v5.0.10` |
| **反向代理 / 静态资源** | NGINX `v2.22.0` |
| **Android 客户端** | Kotlin (`Android 10+`) |
| **Windows 客户端** | C# (`.NET 6+`, `Windows 10+`) |
| **Web 前端** | Vue 3 + TypeScript + Element Plus |
